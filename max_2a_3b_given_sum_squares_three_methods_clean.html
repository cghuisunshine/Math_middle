一、题目1. Problem

题目原图：若 a²+b²=5，求 2a+3b 的最大值。 Original image: If a²+b²=5, find the maximum value of 2a+3b.
已知：a² + b² = 5 求：2a + 3b 的最大值

二、解法一：柯西不等式2. Solution 1: Cauchy-Schwarz Inequality

把 2a+3b 看成两个向量的点乘：View 2a+3b as the dot product of two vectors:

2a + 3b = (2,3) · (a,b)

由柯西不等式：By Cauchy-Schwarz:

(2a + 3b)² ≤ (2²+3²)(a²+b²)

代入 a²+b²=5：Substitute a²+b²=5:

(2a + 3b)² ≤ 13 × 5 = 65

所以：Therefore:

2a + 3b ≤ √65
最大值：√65Maximum value: √65

三、什么时候取到最大值？3. When Is the Maximum Reached?

柯西不等式取等号时，两个向量方向相同：Equality in Cauchy-Schwarz occurs when the two vectors have the same direction:

(a,b) = k(2,3)

设 a=2k，b=3k，代入 a²+b²=5：Let a=2k and b=3k. Substitute into a²+b²=5:

(2k)² + (3k)² = 5 13k² = 5 k = √(5/13)

取正的 k 时，2a+3b 最大：Taking the positive k gives the maximum:

a = 2√(5/13) b = 3√(5/13)

此时：Then:

2a + 3b = 2·2√(5/13) + 3·3√(5/13) = 13√(5/13) = √65

四、解法二：只用 x²+y²≥2xy4. Solution 2: Using Only x²+y²≥2xy

由：From:

x² + y² ≥ 2xy

令 x=3a，y=2b，得到：Let x=3a and y=2b. Then:

(3a)² + (2b)² ≥ 2(3a)(2b)

也就是：That is:

9a² + 4b² ≥ 12ab

现在平方目标式：Now square the target expression:

(2a + 3b)² = 4a² + 12ab + 9b²

用 12ab ≤ 9a² + 4b² 代入：Use 12ab ≤ 9a² + 4b²:

(2a + 3b)² ≤ 4a² + (9a² + 4b²) + 9b² = 13a² + 13b² = 13(a²+b²)

由题设 a²+b²=5，所以：Since a²+b²=5:

(2a + 3b)² ≤ 13 × 5 = 65

因此：Therefore:

2a + 3b ≤ √65

当 3a=2b 且 a、b 与 2a+3b 同号取正方向时可以取等号，所以最大值为：Equality can be reached when 3a=2b and a,b take the positive direction for 2a+3b. Therefore the maximum is:

√65
最大值：√65Maximum value: √65

五、解法三：几何理解5. Solution 3: Geometric Interpretation

条件 a²+b²=5 表示点 (a,b) 在半径为 √5 的圆上。The condition a²+b²=5 means that point (a,b) lies on a circle of radius √5.

2a+3b 是向量 (a,b) 在方向 (2,3) 上的投影乘以 |(2,3)|。当 (a,b) 和 (2,3) 同方向时，值最大。The expression 2a+3b is related to the projection of (a,b) in the direction (2,3). It is maximized when (a,b) has the same direction as (2,3).

最大值 = |(a,b)| · |(2,3)| = √5 · √13 = √65

六、最终答案6. Final Answer

max(2a+3b) = √65
最终答案：√65Final answer: √65
