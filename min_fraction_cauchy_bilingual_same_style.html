一、题目1. Problem

题目原图Original problem image
\[ x>0,\quad y>0 \]
\[ \text{求 }\frac{x+\sqrt{x^2+y^2}}{\sqrt3x+y}\text{ 的最小值。} \]

二、答案2. Answer

\[ \boxed{\frac{\sqrt3}{2}} \]
最小值为 \(\boxed{\dfrac{\sqrt3}{2}}\)。The minimum value is \(\boxed{\dfrac{\sqrt3}{2}}\).

三、方法一：先直接猜出最小值，再用柯西证明3. Method 1: Guess the Value First, Then Prove It with Cauchy

我们想证明：We want to prove:

\[ \frac{x+\sqrt{x^2+y^2}}{\sqrt3x+y}\ge \frac{\sqrt3}{2} \]

也就是：That is equivalent to:

\[ x+\sqrt{x^2+y^2}\ge \frac{\sqrt3}{2}(\sqrt3x+y) \]

展开右边：Expand the right-hand side:

\[ \frac{\sqrt3}{2}(\sqrt3x+y)=\frac32x+\frac{\sqrt3}2y \]

于是只需证明：So it is enough to prove:

\[ \sqrt{x^2+y^2}\ge \frac12x+\frac{\sqrt3}2y \]

注意到：Notice that:

\[ \left(\frac12\right)^2+\left(\frac{\sqrt3}2\right)^2=1 \]

由柯西不等式（或向量点积不超过模长乘积）：By Cauchy’s inequality (or the dot-product form):

\[ \frac12x+\frac{\sqrt3}2y \le \sqrt{x^2+y^2}\cdot \sqrt{\left(\frac12\right)^2+\left(\frac{\sqrt3}2\right)^2} =\sqrt{x^2+y^2} \]

因此：Therefore:

\[ x+\sqrt{x^2+y^2} \ge x+\frac12x+\frac{\sqrt3}2y =\frac{\sqrt3}2(\sqrt3x+y) \]
\[ \frac{x+\sqrt{x^2+y^2}}{\sqrt3x+y}\ge \frac{\sqrt3}2 \]
所以原式的最小值不小于 \(\dfrac{\sqrt3}2\)。So the expression is at least \(\dfrac{\sqrt3}2\).

四、方法二：如果不知道答案，先设最小值为 \(k\)4. Method 2: If the Value Is Unknown, Set the Minimum to \(k\)

诀窍：不要先猜 \(\dfrac{\sqrt3}2\)。先设最小值是 \(k\)，再反推柯西不等式所需要的系数。Key trick: do not guess \(\dfrac{\sqrt3}2\) first. Set the minimum to \(k\), then reverse-engineer the coefficients needed for Cauchy’s inequality.

希望证明：We hope to prove:

\[ \frac{x+\sqrt{x^2+y^2}}{\sqrt3x+y}\ge k \]

等价于：This is equivalent to:

\[ x+\sqrt{x^2+y^2}\ge k(\sqrt3x+y) \]

移项：Rearrange:

\[ \sqrt{x^2+y^2}\ge (k\sqrt3-1)x+ky \]

把根号看成向量 \((x,y)\) 的长度：View the radical as the length of the vector \((x,y)\):

\[ \sqrt{x^2+y^2} \]

柯西不等式告诉我们：Cauchy’s inequality tells us:

\[ ux+vy\le \sqrt{x^2+y^2} \quad\text{whenever}\quad u^2+v^2=1 \]

我们希望右边恰好是 \((k\sqrt3-1)x+ky\)，所以取：We want the right-hand side to match \((k\sqrt3-1)x+ky\), so choose:

\[ u=k\sqrt3-1,\qquad v=k \]

于是必须满足：Then we must have:

\[ (k\sqrt3-1)^2+k^2=1 \]

展开：Expand:

\[ 3k^2-2\sqrt3k+1+k^2=1 \]
\[ 4k^2-2\sqrt3k=0 \]
\[ 2k(2k-\sqrt3)=0 \]

因为最小值应为正数，所以：Since the minimum value should be positive, we get:

\[ k=\frac{\sqrt3}2 \]

这就反推出柯西不等式中的系数应为：So this reverse-engineers the coefficients in Cauchy’s inequality:

\[ u=k\sqrt3-1=\frac12,\qquad v=k=\frac{\sqrt3}2 \]

于是：Hence:

\[ \sqrt{x^2+y^2}\ge \frac12x+\frac{\sqrt3}2y \]
\[ x+\sqrt{x^2+y^2}\ge x+\frac12x+\frac{\sqrt3}2y=\frac{\sqrt3}2(\sqrt3x+y) \]
\[ \frac{x+\sqrt{x^2+y^2}}{\sqrt3x+y}\ge \frac{\sqrt3}2 \]
因此，原式最小值就是 \(\boxed{\dfrac{\sqrt3}2}\)。Therefore, the minimum value is \(\boxed{\dfrac{\sqrt3}2}\).

五、等号成立条件5. Equality Condition

柯西不等式取等号时，两个向量同方向，因此：Equality in Cauchy occurs when the two vectors are in the same direction. Hence:

\[ (x,y)\parallel \left(\frac12,\frac{\sqrt3}2\right) \]
\[ \frac{y}{x}=\sqrt3 \]

也就是：That is:

\[ y=\sqrt3x \]

例如取 \(x=1,\ y=\sqrt3\)，则：For example, take \(x=1,\ y=\sqrt3\). Then:

\[ \frac{x+\sqrt{x^2+y^2}}{\sqrt3x+y} =\frac{1+\sqrt{1+3}}{\sqrt3+\sqrt3} =\frac3{2\sqrt3} =\frac{\sqrt3}2 \]
所以最小值可以取到。So the minimum value is attainable.

六、最终答案6. Final Answer

\[ \boxed{\min\frac{x+\sqrt{x^2+y^2}}{\sqrt3x+y}=\frac{\sqrt3}2} \]
