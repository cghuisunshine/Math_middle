Chapter 1. Algebraic Expressions

第1章 代数式

English

An algebraic expression uses numbers, variables, and operations without an equals sign.

3x + 5, 2a - 7, (x+1)/2

Learn how to substitute values and simplify expressions by following order of operations.

Mini practice: If x=4, find 2x+3.

3x + 5、2a - 7、(x+1)/2

这一章学习代入求值，以及按正确运算顺序化简式子。

小练习：若 x=4，求 2x+3。

Chapter 2. Equations

第2章 方程

English

An equation states that two expressions are equal. Solving means finding the value that makes the statement true.

x + 4 = 9 gives x = 5.

Use balance thinking: do the same operation to both sides.

Mini practice: Solve 3x = 15.
中文

方程表示两个式子相等。解方程，就是找出使等式成立的未知数值。

x + 4 = 9，所以 x = 5。

用“天平思想”：对等号两边做同样的操作。

小练习：解 3x = 15。

English

Number theory studies integers, factors, multiples, primes, divisibility, greatest common factor, and least common multiple.

These ideas help simplify fractions and solve structured problems.

12 的因数：1,2,3,4,6,12。质数：2,3,5,7,11,...

这些知识可以帮助我们约分、通分和解决有规律的题目。

几何常常把图形观察和代数推理结合起来。

第5章 基础应用题

English

Translate words into equations. Identify what is known, what is unknown, and what relationship connects them.

“A number plus 7 is 12” becomes x + 7 = 12.

Careful reading is usually more important than long calculation.

Mini practice: “Twice a number is 18.” Write and solve an equation.

把文字信息翻译成方程。先找已知量、未知量，以及它们之间的关系。

“一个数加 7 等于 12”可写成 x + 7 = 12。

应用题中，认真读题往往比长计算更重要。

小练习：“一个数的两倍等于 18。”列式并求解。

Chapter 6. Linear Equations in One Variable

第6章 一元一次方程

English

These equations have one variable and the highest power is 1.

2x + 3 = 11 → 2x = 8 → x = 4

Combine like terms, clear fractions carefully, and isolate the variable step by step.

Mini practice: Solve 5x - 7 = 18.
中文

一元一次方程只含一个未知数，且未知数的最高次数是 1。

2x + 3 = 11 → 2x = 8 → x = 4

注意合并同类项、去分母，并一步一步把变量单独移出来。

小练习：解 5x - 7 = 18。

Chapter 7. Linear Equations in Two Variables

第7章 二元一次方程

English

An equation like 2x + y = 7 has infinitely many solutions, each represented by an ordered pair.

If x = 2, then y = 3 is one solution.

This chapter prepares for graphing and systems.

Mini practice: Is (1,5) a solution to 2x+y=7?
中文

像 2x + y = 7 这样的方程含有两个变量，通常有无穷多组解，每组解是一个有序数对。

如果 x = 2，那么 y = 3 是其中一组解。

这一章为后面画图和解方程组打基础。

小练习：(1,5) 是 2x+y=7 的解吗？

Chapter 8. Ratios, Proportions, and Percents

第8章 比、比例与百分数

English

Ratios compare quantities. Proportions state two ratios are equal. Percents mean “per hundred.”

3/5 = 6/10, and 25% = 25/100 = 0.25

These topics appear in discounts, taxes, growth, and mixture problems.

Mini practice: Find 20% of 60.
中文

“比”用来比较两个量；“比例”表示两个比相等；百分数就是“每一百份中的多少”。

3/5 = 6/10，且 25% = 25/100 = 0.25

这些知识常见于折扣、税率、增长率和混合问题。

小练习：求 60 的 20%。

Linear equations graph as straight lines. Common forms include slope-intercept form y = mx + b.

Chapter 10. Systems of Equations

第10章 方程组

English

A system asks for values that satisfy two or more equations at the same time.

Solve by substitution or elimination:
x + y = 5
x - y = 1

The solution is the intersection point of the graphs.

Mini practice: Solve the system above.
中文

方程组要求找到同时满足两个或多个方程的数值。

可用代入法或消元法：
x + y = 5
x - y = 1

从图像上看，解就是两条直线的交点。

小练习：解上面的方程组。

x + 3 < 8 可得 x < 5。

重要规则：两边同乘或同除一个负数时，不等号方向要改变。

Factoring rewrites an expression as a product.

Common methods include GCF, grouping, and pattern recognition such as difference of squares.

Mini practice: Factor x² - 9.

x² + 5x + 6 = (x+2)(x+3)

常见方法有提公因式、分组分解、公式识别，如平方差公式。

Chapter 15. Quadratic Equations

第15章 二次方程

English

Quadratic equations have degree 2. Solve them by factoring, completing the square, or using the quadratic formula.

x² - 5x + 6 = 0 gives x=2 or x=3.

Check whether the equation can be factored first before using a longer method.

Mini practice: Solve x² - 4 = 0.
中文

二次方程的最高次数是 2。常用解法有因式分解法、配方法和求根公式。

x² - 5x + 6 = 0 的解是 x=2 或 x=3。

遇到二次方程时，先看看能不能直接分解会比较快。

小练习：解 x² - 4 = 0。

解根式方程时，通常先把根式单独留一边，再两边平方，最后一定要验根。

Chapter 18. Functions

第18章 函数

English

A function assigns exactly one output to each input in its domain.

f(x)=2x+1; then f(3)=7.

Study notation, domain, range, and how functions describe rules.

Mini practice: If f(x)=x², find f(4).

f(x)=2x+1，那么 f(3)=7。

这一章学习函数记号、定义域、值域，以及如何把规则写成函数。

应先掌握指数运算规则，再学习如何用对数解指数方程。
