Introduction to Algebra — Bilingual Tutorial

代数入门——双语教程

A beginner-friendly handout in English and Chinese, based on the introductory algebra ideas: variables, expressions, equations, properties, and simple solving.

这是一个适合初学者的英汉双语讲义，内容包括：变量、式子、方程、运算性质，以及简单方程的解法。

1. What is Algebra?

1. 什么是代数？

English

Algebra is a way of using letters and symbols to represent numbers.

Example: In arithmetic we write 3 + 5 = 8. In algebra we may write x + 5 = 8.

Here, x stands for a number we do not yet know.

Explanation

Algebra helps us describe patterns and rules. Instead of solving only one example, we can solve many similar problems with one idea.

中文

代数是一种用字母和符号来表示数字的方法。

例子：在算术里我们写 3 + 5 = 8；在代数里我们可以写 x + 5 = 8。

这里的 x 表示一个我们暂时还不知道的数。

解释

代数可以帮助我们描述规律和规则。它不只是解决一个具体题目，而是用一个方法解决很多相似的问题。

2. Variables

2. 变量

English

A variable is a letter that represents a number.

  • x, y, a, n
If x = 4, then x + 3 = 7.
If y = 10, then 2y = 20.

Variables are like placeholders for numbers.

中文

变量是代表数字的字母。

  • x、y、a、n
如果 x = 4，那么 x + 3 = 7。
如果 y = 10，那么 2y = 20。

变量就像数字的占位符。

3. Types of Numbers

3. 数的类型

English
  • Whole numbers: 0, 1, 2, 3, ...
  • Integers: ..., -2, -1, 0, 1, 2, ...
  • Rational numbers: numbers that can be written as fractions, like 1/2 or -3/4

Important fact: 0 is neither positive nor negative.

中文
  • 全体整数中的非负数（whole numbers）：0，1，2，3，……
  • 整数（integers）：……，-2，-1，0，1，2，……
  • 有理数（rational numbers）：可以写成分数的数，例如 1/2、-3/4

重要事实：0 既不是正数，也不是负数。

4. Order of Operations

4. 运算顺序

English
  1. Parentheses
  2. Exponents
  3. Multiplication and division, left to right
  4. Addition and subtraction, left to right
Example: 2 + 3 × 4 = 2 + 12 = 14
Example: (2 + 3) × 4 = 5 × 4 = 20

Parentheses can completely change the answer.

中文
  1. 先算括号
  2. 再算乘方
  3. 再算乘除，从左到右
  4. 最后算加减，从左到右
例子：2 + 3 × 4 = 2 + 12 = 14
例子：(2 + 3) × 4 = 5 × 4 = 20

括号会明显改变答案，所以要先看有没有括号。

5. Important Properties

5. 重要运算性质

English

Commutative Property

a + b = b + a, and ab = ba

But subtraction and division are not commutative.

Associative Property

(a + b) + c = a + (b + c)

(ab)c = a(bc)

Distributive Property

a(b + c) = ab + ac

3(x + 2) = 3x + 6
中文

交换律

a + b = b + a，以及 ab = ba

但减法和除法不满足交换律。

结合律

(a + b) + c = a + (b + c)

(ab)c = a(bc)

分配律

a(b + c) = ab + ac

3(x + 2) = 3x + 6

6. Expressions and Equations

6. 式子与方程

English

An expression is a math phrase without an equals sign.

  • x + 5
  • 3y - 2

An equation says two things are equal.

  • x + 3 = 7
  • 2y = 10

Think of an equation like a balance scale.

中文

式子（expression）是没有等号的数学表达。

  • x + 5
  • 3y - 2

方程（equation）表示两边相等。

  • x + 3 = 7
  • 2y = 10

可以把方程想象成一个天平，两边必须保持平衡。

7. Solving Simple Equations

7. 解简单方程

English
Example 1: x + 4 = 9
Subtract 4 from both sides: x = 5
Example 2: 3x = 12
Divide both sides by 3: x = 4
Example 3: x / 5 = 7
Multiply both sides by 5: x = 35

The goal is to get the variable alone.

中文
例1：x + 4 = 9
两边都减去 4：x = 5
例2：3x = 12
两边都除以 3：x = 4
例3：x / 5 = 7
两边都乘以 5：x = 35

目标是把变量单独留在一边。

8. Combining Like Terms

8. 合并同类项

English

Like terms have the same variable part.

  • 3x + 5x = 8x
  • 7y - 2y = 5y

But these cannot be combined:

  • 3x + 4
  • 2x + 3y
中文

同类项必须有相同的字母部分。

  • 3x + 5x = 8x
  • 7y - 2y = 5y

下面这些不能合并：

  • 3x + 4
  • 2x + 3y

9. Exponents and Fractions

9. 乘方与分数

English

Exponents: x² = x × x, x³ = x × x × x

Fractions: To add fractions, use a common denominator.

1/4 + 2/4 = 3/4

Warning: Never divide by 0.

中文

乘方：x² = x × x，x³ = x × x × x

分数：分数相加时，要先通分。

1/4 + 2/4 = 3/4

注意：绝对不能除以 0。
