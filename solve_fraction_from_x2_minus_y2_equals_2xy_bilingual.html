一、题目1. Problem

题目原图 Original problem image
\[ x^2-y^2=2xy \]
\[ \text{求 }\frac{x+y}{x-y} \]

二、答案2. Answer

\[ \boxed{1+\sqrt2\ \text{或}\ 1-\sqrt2} \]
当 \(x,y\) 不限制正负时，结果有两个可能值：\(1+\sqrt2\) 或 \(1-\sqrt2\)。 When \(x,y\) are not restricted to be positive, there are two possible values: \(1+\sqrt2\) or \(1-\sqrt2\).
因此这里不能舍去 \(1-\sqrt2\)。 Therefore, we should not reject \(1-\sqrt2\) here.

三、方法一：设目标式为 \(t\)3. Method 1: Let the Target Fraction Be \(t\)

我们要求的是： We need to find:

\[ \frac{x+y}{x-y} \]

所以直接设： So directly let:

\[ t=\frac{x+y}{x-y} \]

设 Let

\[ A=x+y,\qquad B=x-y \]

那么目标式就是： Then the target fraction is:

\[ t=\frac AB \]

四、把已知条件改写成 \(A,B\)4. Rewrite the Given Condition in Terms of \(A,B\)

因为： Because:

\[ x^2-y^2=(x+y)(x-y)=AB \]

而： Also:

\[ x=\frac{A+B}{2},\qquad y=\frac{A-B}{2} \]

所以： Therefore:

\[ xy=\frac{A+B}{2}\cdot\frac{A-B}{2} =\frac{A^2-B^2}{4} \]

代入已知条件 \(x^2-y^2=2xy\)： Substitute into the given condition \(x^2-y^2=2xy\):

\[ AB=2\cdot\frac{A^2-B^2}{4} \]
\[ AB=\frac{A^2-B^2}{2} \]
\[ 2AB=A^2-B^2 \]

五、两边除以 \(B^2\)5. Divide Both Sides by \(B^2\)

因为要求 \(\frac{x+y}{x-y}\)，所以 \(x-y\ne0\)，也就是 \(B\ne0\)。 Since the fraction \(\frac{x+y}{x-y}\) is required, \(x-y\ne0\), so \(B\ne0\).

由： From:

\[ 2AB=A^2-B^2 \]

两边除以 \(B^2\)： Divide both sides by \(B^2\):

\[ 2\frac AB=\left(\frac AB\right)^2-1 \]

由于 \(t=\frac AB\)，所以： Since \(t=\frac AB\), we get:

\[ 2t=t^2-1 \]
\[ t^2-2t-1=0 \]

六、解关于 \(t\) 的方程6. Solve the Equation for \(t\)

\[ t^2-2t-1=0 \]

用配方法： Complete the square:

\[ t^2-2t=1 \]
\[ (t-1)^2=2 \]
\[ t=1\pm\sqrt2 \]
也就是说： That is: \[ \frac{x+y}{x-y}=1\pm\sqrt2 \]

七、不限制正负时保留两个值7. Keep Both Values Without Sign Restrictions

前面已经得到： From the previous step, we have:

\[ \frac{x+y}{x-y}=1\pm\sqrt2 \]

如果题目说明 \(x,y>0\)，因为 \(\frac{x+y}{x-y}>0\)，才会舍去负值 \(1-\sqrt2\)。 If the problem states \(x,y>0\), then \(\frac{x+y}{x-y}>0\), so the negative value \(1-\sqrt2\) would be rejected.

但现在 \(x,y\) 不限制正负，所以两个值都可以保留。 But now \(x,y\) are not restricted by sign, so both values must be kept.

因此结果是两个可能值：\(1+\sqrt2\) 和 \(1-\sqrt2\)。 Therefore, the result has two possible values: \(1+\sqrt2\) and \(1-\sqrt2\).

八、最终答案8. Final Answer

\[ \boxed{1+\sqrt2\ \text{或}\ 1-\sqrt2} \]
当 \(x,y\) 不限正负时，最终答案是两个可能值。 When \(x,y\) are not restricted by sign, the final answer has two possible values.
答案：\(1+\sqrt2\) 或 \(1-\sqrt2\)。 Answer: \(1+\sqrt2\) or \(1-\sqrt2\).

九、方法二：设 \(t=\frac{x}{y}\)9. Method 2: Let \(t=\frac{x}{y}\)

方法二手写思路图 Handwritten outline of Method 2

这一种方法更直接：先把已知式两边都除以 \(y^2\)，再设 \(t=\frac{x}{y}\)。 This method is more direct: divide both sides by \(y^2\), then let \(t=\frac{x}{y}\).

\[ x^2-y^2=2xy \]

两边除以 \(y^2\)： Divide both sides by \(y^2\):

\[ \frac{x^2}{y^2}-1=\frac{2xy}{y^2}=2\cdot \frac{x}{y} \]

设 Let

\[ t=\frac{x}{y} \]

那么得到： Then we get:

\[ t^2-1=2t \]
\[ t^2-2t-1=0 \]
\[ (t-1)^2=2 \]
\[ t=1\pm\sqrt2 \]

十、代回目标式10. Substitute Back into the Target Fraction

因为 \(t=\frac{x}{y}\)，所以： Since \(t=\frac{x}{y}\), we have:

\[ \frac{x+y}{x-y}=\frac{\frac{x}{y}+1}{\frac{x}{y}-1}=\frac{t+1}{t-1} \]

分别代入两个值： Substitute the two possible values:

\[ t=1+\sqrt2\Rightarrow \frac{t+1}{t-1}=\frac{2+\sqrt2}{\sqrt2}=1+\sqrt2 \]
\[ t=1-\sqrt2\Rightarrow \frac{t+1}{t-1}=\frac{2-\sqrt2}{-\sqrt2}=1-\sqrt2 \]
所以目标式的可能值仍然是： So the possible values of the target expression are still: \[ \frac{x+y}{x-y}=1\pm\sqrt2 \]
由于现在 \(x,y\) 不限正负，所以两个值都保留：\(1+\sqrt2\) 或 \(1-\sqrt2\)。 Since \(x,y\) are not restricted by sign here, keep both values: \(1+\sqrt2\) or \(1-\sqrt2\).
