专题1.1 同底数幂的乘法

  1. 题型01 同底数幂相乘 计算：a5 · a3 · a。
  2. 题型02 不同底数转化为同底数幂的运算 计算：26 · 42 · 8，并把结果写成 2n 的形式。
  3. 题型03 同底数幂乘法的逆用 已知 2m = 8，2n = 16，求 2m+n 的值。
  4. 题型04 已知代数式的值，求式子的值 已知 xm = 3，xn = 5，求 2xm+n 的值。
  5. 题型05 新定义有关同底数幂的运算 定义 a ★ b = 3a · 3b，求 2 ★ 4，并写成 3 的幂。
  6. 题型06 与同底数幂运算有关的规律探究问题 观察：31，32，33，34，...。若 A = 32 · 35 · 38，请写出 A 的指数，并说明指数的规律。

专题1.2 幂的乘方与积的乘方

  3. 题型03 利用幂的乘方比较大小 比较 512 与 256 的大小，并说明理由。
  4. 题型04 积的乘方运算 计算：(-2a2b)3。
  5. 题型05 积的乘方的逆用 已知 xy = -3，求 (2xy)3 的值。

专题1.4 整式的乘法

  1. 题型01 计算单项式乘单项式 计算：(-3a2b)(4ab3)。
  2. 题型02 计算单项式乘多项式 计算：2x(3x2 - 4x + 5)。
  3. 题型03 计算多项式乘多项式 计算：(x + 3)(x - 5)。
  4. 题型04 已知多项式乘积不含某项求字母的值 若 (x + 2)(x + k) 的展开式中不含一次项，求 k。
  5. 题型05 整式乘法混合运算 化简：(x + 2)(x - 3) - x(x - 1)。
  6. 题型06 多项式乘多项式一化简求值 先化简，再求值：(x + 1)(x - 2) + (x - 3)2，其中 x = 2。
  7. 题型07 (x+p)(x+q)型多项式乘法 计算：(x + 4)(x - 7)。
  8. 题型08 多项式乘法中的规律性问题 观察 (x - 1)(x + 1) = x2 - 1。利用这个规律计算：(x - 1)(x + 1)(x2 + 1)。
  9. 题型09 整式乘法与图形面积 一个长方形的长为 2x + 3，宽为 x + 4。用含 x 的式子表示它的面积。
  10. 题型10 整式运算中的新定义型问题 定义 f(a,b) = (x + a)(x + b)。求 f(2,-5) 的展开式。

专题1.5 平方差公式

  1. 题型01 判断是否可用平方差公式运算 判断 (2x + 3)(2x - 3) 能否用平方差公式计算；若能，请直接写出结果。
  2. 题型02 运用平方差公式进行运算 计算：(5a - 2b)(5a + 2b)。
  3. 题型03 利用平方差公式进行简便运算 用简便方法计算：1002 × 998。
  4. 题型04 与平方差公式有关的化简求值 先化简，再求值：(x + y)(x - y) + y2，其中 x = 5，y = 3。
  5. 题型05 利用平方差公式求解连乘问题 计算：(2 + 1)(22 + 1)(24 + 1)(28 + 1)。
  6. 题型06 平方差公式在几何图形中的应用 一个大正方形边长为 x + 6，小正方形边长为 x - 6。用平方差公式表示两个正方形面积之差。

专题1.6 完全平方公式

  1. 题型01 判断是否完全平方公式运算 判断 9x2 + 12x + 4 是否为完全平方式；若是，请写成平方的形式。
  2. 题型02 运用完全平方公式进行运算 计算：(3a - 2b)2。
  3. 题型03 求完全平方式中的字母系数 若 x2 + kx + 25 是完全平方式，求 k 的值。
  4. 题型04 平方差与完全平方综合进行运算 化简：(x + 3)2 - (x - 3)2。
  5. 题型05 利用完全平方公式进行简便运算 用简便方法计算：1012。
  6. 题型06 与乘法公式有关的化简求值问题 先化简，再求值：(a + b)2 - (a - b)2，其中 a = 7，b = 2。
  7. 题型07 通过对完全平方公式变形求值 已知 x + 1x = 5，求 x2 + 1x2 的值。
  8. 题型08 利用完全平方式求代数式的最值问题 求代数式 x2 - 6x + 11 的最小值。
  9. 题型09 完全平方公式在几何图形中的应用 一个边长为 a + b 的正方形，被分成边长分别含 a、b 的小区域。请用面积关系写出 (a + b)2 的展开式。

专题1.7 整式的除法

  1. 题型01 单项式除以单项式 计算：12a5b3 ÷ (-3a2b)。
  2. 题型02 多项式除以单项式 计算：(6x3 - 9x2 + 3x) ÷ 3x。
  3. 题型03 含整式除法的整式四则混合运算 化简：[(-2a2b)2 · (3ab)] ÷ (6a3b2)。
  4. 题型04 整式的混合运算之化简求值 先化简，再求值：[(x3 - 2x2 + x) ÷ x] - (x - 1)2，其中 x = 4。
  5. 题型05 含整式除法的新定义型问题 定义 P ⊘ m 表示把多项式 P 的每一项都除以单项式 m。计算：(6a3b - 9a2b2) ⊘ (3ab)。
  6. 题型06 利用竖式的方法求整式中多项式除以单项式 用竖式或验算的方法计算：(x2 + 5x + 6) ÷ (x + 2)。
