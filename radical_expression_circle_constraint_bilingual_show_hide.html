一、题目1. Problem

题目原图：若 x²+y²=16，求两个根式之和。Original image: Given x²+y²=16, find the sum of the two radicals.
已知：x² + y² = 16 求：√(y² - 8y + 16) + √(xy - 5y + 4x - 20)

二、关键：第二个根号限制范围2. Key: The Second Radical Restricts the Values

先把第二个根号内因式分解：Factor the expression inside the second radical:

xy - 5y + 4x - 20 = y(x-5) + 4(x-5) = (x-5)(y+4)

由 x²+y²=16 可知：From x²+y²=16, we know:

-4 ≤ x ≤ 4, -4 ≤ y ≤ 4

所以 x-5 < 0，且 y+4 ≥ 0。根号内必须 ≥0：So x-5 < 0 and y+4 ≥ 0. The expression under the square root must be ≥0:

(x-5)(y+4) ≥ 0

只能有：Therefore we must have:

y+4=0 y=-4

三、代入计算3. Substitute and Calculate

把 y=-4 代入 x²+y²=16：Substitute y=-4 into x²+y²=16:

x²+(-4)²=16 x²+16=16 x=0

原式：The original expression:

√[(-4)² - 8(-4) + 16] + √[(0-5)(-4+4)] = √64 + √0 = 8
答案：8Answer: 8

四、最终答案4. Final Answer

√(y² - 8y + 16) + √(xy - 5y + 4x - 20) = 8
最终答案：8Final answer: 8
