三、方法一：设值法 + 二次方程判别式3. Method 1: Set the Value and Use the Discriminant

设原式等于 \(k\)： Set the expression equal to \(k\):

\[ \frac{x}{(x+1)^2}=k \]

两边乘以 \((x+1)^2\)： Multiply both sides by \((x+1)^2\):

\[ x=k(x+1)^2 \]
\[ x=kx^2+2kx+k \]

整理成关于 \(x\) 的二次方程： Rearrange it into a quadratic equation in \(x\):

\[ kx^2+(2k-1)x+k=0 \]

如果某个 \(k\) 能成为原式的值，那么这个方程必须有实数解，所以判别式满足： If a value \(k\) is attainable, this equation must have a real solution, so the discriminant must satisfy:

\[ \Delta=(2k-1)^2-4k^2\ge0 \]
\[ 4k^2-4k+1-4k^2\ge0 \]
\[ 1-4k\ge0 \]
\[ k\le\frac14 \]
所以原式的值不可能超过 \(\frac14\)。 Therefore, the expression cannot exceed \(\frac14\).

接着验证 \(k=\frac14\) 能取到： Now verify that \(k=\frac14\) is attainable:

\[ \frac{x}{(x+1)^2}=\frac14 \]
\[ 4x=(x+1)^2 \]
\[ x^2-2x+1=0 \]
\[ (x-1)^2=0 \]
\[ x=1 \]
因此最大值为 \(\boxed{\frac14}\)，当 \(x=1\) 时取得。 Thus the maximum value is \(\boxed{\frac14}\), attained when \(x=1\).

四、方法二：利用 \(2x\le x^2+1\)4. Method 2: Use \(2x\le x^2+1\)

因为平方非负： Since a square is nonnegative:

\[ (x-1)^2\ge0 \]
\[ x^2-2x+1\ge0 \]
\[ 2x\le x^2+1 \]

两边同时加上 \(2x\)： Add \(2x\) to both sides:

\[ 4x\le x^2+2x+1 \]
\[ 4x\le (x+1)^2 \]

因为 \((x+1)^2>0\)，两边除以 \(4(x+1)^2\)： Since \((x+1)^2>0\), divide both sides by \(4(x+1)^2\):

\[ \frac{x}{(x+1)^2}\le\frac14 \]

当 \(x=1\) 时，\((x-1)^2=0\)，等号成立。 When \(x=1\), \((x-1)^2=0\), so equality holds.

所以最大值为 \(\boxed{\frac14}\)。 Therefore, the maximum value is \(\boxed{\frac14}\).
