为什么旋转后，上界会变小？

旋转并没有改变原题。它只是帮我们更好地使用隐藏条件：

∠EMF = 120°

1. 不旋转时，上界太松

如果只看原来的两个圆，那么可以得到一个粗略上界：

EF ≤ AE + AB + BF = 2 + 8 + 8 = 18

但是这个估计忽略了角度条件。要达到 18，点大致要排成一条直线，此时角度更接近 180°，不是 120°。所以 18 是一个不可能达到的松上界。

2. 旋转把 120° 条件用进来了

把 E 绕 M 旋转 120° 得到 G。因为旋转保持距离和角度，所以：

ME = MG，  MF 是公共边，  ∠EMF = ∠GMF

因此两个三角形全等：

△EMF ≅ △GMF   ⇒   EF = GF

3. 圆心从 A 变成 C

旋转后，E 变成 G，同时原来的圆心 A 也变成 C：

E ∈ circle(A, 2)   ⇒   G ∈ circle(C, 2)

现在关键的圆心距离不再是 AB，而是 CB：

AB = 8，但是旋转后 CB = 4

4. 所以新的上界更小

现在只需要在 circle(C,2) 和 circle(B,8) 之间最大化 GF：

GF ≤ 2 + CB + 8 = 2 + 4 + 8 = 14

因为 EF = GF，所以：

EF ≤ 14

核心：旋转不是“神奇地缩短距离”，而是把原来隐藏的 120° 条件变成了新的固定点 C。原来忽略角度时只能得到松上界 18；旋转后角度被编码到 C 的位置中，于是更紧的上界 14 就出现了。

Why does rotation make the upper bound smaller?

Rotation does not change the original problem. It gives us a better way to use the hidden condition:

∠EMF = 120°

1. Without rotation, the upper bound is too loose

If we only use the two original circles, then a rough upper bound is:

EF ≤ AE + AB + BF = 2 + 8 + 8 = 18

But this ignores the angle condition. The value 18 would happen when the points are almost in one straight line, which would make the angle closer to 180°, not 120°. So 18 is an impossible loose bound.

2. Rotation uses the 120° condition

Rotate E around M by 120° to get G. Because rotation preserves distance and angle, we get:

ME = MG,   MF is common,   ∠EMF = ∠GMF

So the two triangles are congruent:

△EMF ≅ △GMF   ⇒   EF = GF

3. The circle center changes from A to C

After rotation, E becomes G, and the original circle centered at A becomes a new circle centered at C:

E ∈ circle(A, 2)   ⇒   G ∈ circle(C, 2)

Now the important distance is not AB anymore. It is CB:

AB = 8, but after rotation CB = 4

In the animation, Step 6 first shows a general position G′ on circle(C,2) and F′ on circle(B,8), and then Step 7 moves them into the extremal collinear position. This makes the transformation easier to see.

4. Therefore the new upper bound is smaller

Now we only need to maximize GF between circle(C, 2) and circle(B, 8):

GF ≤ 2 + CB + 8 = 2 + 4 + 8 = 14

Since EF = GF, we get:

EF ≤ 14

Key idea: rotation does not magically shrink the same distance. It changes a loose bound that ignored the angle, 18, into a tighter bound that uses the 120° angle, 14.

本质：为什么旋转能得到更紧的上界？

旋转不只是把图形转一下。它的本质是：把角度条件转化成距离问题。

1. 真正困难的是固定角

∠EMF = 120°

如果只看原来的两个圆：

E ∈ circle(A,2)，  F ∈ circle(B,8)

得到的粗略上界是：

EF ≤ AE + AB + BF = 2 + 8 + 8 = 18

但它没有用到 120°，所以太大。

2. 旋转正好使用 120°

把 E 绕 M 旋转 120° 得到 G。旋转保持长度，因此：

ME = MG

MF 又是公共边。因为旋转角正好对应题目给出的 120°，所以：

△EMF ≅ △GMF

于是：

EF = GF

这一步是关键：我们不再直接最大化 EF，而是最大化与它相等的 GF。

3. A 旋转到 C，所以圆心变了

E 旋转到 G 的同时，圆心 A 也旋转到 C。因此：

circle(A,2) → circle(C,2)

所以旋转后：

G ∈ circle(C,2)，  F ∈ circle(B,8)

现在起控制作用的圆心距不是 AB，而是 CB。

AB = 8，但是 CB = 4

4. 更紧的上界出现了

旋转后的两个圆之间，最大距离为：

GF ≤ 2 + CB + 8 = 2 + 4 + 8 = 14

由于 EF = GF，所以：

EF ≤ 14

一句话总结：旋转让 120° 条件显现为新的固定点 C。不旋转时，角度条件没有进入估计，得到 18；旋转后，角度条件进入了 C 的位置，得到真正有用的上界 14。

In essence: why does rotation make a tighter bound possible?

Rotation is not just moving the picture. It is a way to convert the angle condition into a distance condition.

1. The hard condition is the fixed angle

∠EMF = 120°

If we only look at the original two circles,

E ∈ circle(A,2),   F ∈ circle(B,8)

then the rough upper bound is:

EF ≤ AE + AB + BF = 2 + 8 + 8 = 18

But this ignores the 120° angle, so it is too loose.

2. Rotation uses the 120° angle

Rotate E around M by 120° to get G. Rotation preserves distance, so:

ME = MG

Also, MF is the common side. Because the rotation angle matches the given 120° angle, the two triangles match:

△EMF ≅ △GMF

Therefore:

EF = GF

This is the key transformation: instead of maximizing EF directly, we can maximize the equal segment GF.

3. A rotates to C, so the circle center changes

When E rotates to G, the center A also rotates to C. Therefore:

circle(A,2) → circle(C,2)

So after rotation:

G ∈ circle(C,2),   F ∈ circle(B,8)

Now the important center distance is not AB anymore. It is CB.

AB = 8, but CB = 4

4. The tighter upper bound appears

For the rotated problem, the maximum possible distance between the two circles is:

GF ≤ 2 + CB + 8 = 2 + 4 + 8 = 14

Since EF = GF, we get:

EF ≤ 14

Key idea: rotation makes the 120° condition show up as the new fixed point C. Without rotation, the angle condition is hidden and the loose bound is 18. With rotation, the angle condition is built into the new circle center C, so the tighter bound 14 becomes visible.
