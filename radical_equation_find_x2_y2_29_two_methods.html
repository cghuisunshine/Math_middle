一、题目1. Problem

题目原图：已知 x√(29−y²)+y√(29−x²)=29，求 x²+y²。 Original image: Given x√(29−y²)+y√(29−x²)=29, find x²+y².
已知：x√(29-y²) + y√(29-x²) = 29 求：x² + y²

二、解法：柯西不等式2. Solution: Cauchy-Schwarz Inequality

把左边看成两个向量的点乘：View the left side as the dot product of two vectors:

x√(29-y²) + y√(29-x²)

由柯西不等式：By Cauchy-Schwarz:

[x√(29-y²) + y√(29-x²)]² ≤ (x²+y²)[(29-y²)+(29-x²)]

令：Let:

S = x² + y²

那么：Then:

(29-y²)+(29-x²) = 58 - (x²+y²) = 58 - S

题目给出左边等于 29，所以：The problem states the left side equals 29, so:

29² ≤ S(58-S)

三、利用最大值判断3. Use the Maximum Value

考虑函数：Consider the expression:

S(58-S)

它是一个开口向下的二次式，最大值在 S=29 时取得：It is a downward-opening quadratic expression, and its maximum occurs at S=29:

S(58-S) = 29² - (S-29)² ≤ 29²

但前面又有：But we also have:

29² ≤ S(58-S)

所以只能两边同时取等号：Therefore equality must hold:

S(58-S)=29² S=29
答案：x²+y²=29Answer: x²+y²=29

四、解法二：只用 a²+b²≥2ab4. Solution 2: Using Only a²+b²≥2ab

基本不等式：The basic inequality is:

a² + b² ≥ 2ab

把它变形为：Rewrite it as:

ab ≤ (a²+b²)/2

先看第一项。令：For the first term, let:

a = x b = √(29-y²)

则：Then:

x√(29-y²) ≤ [x² + (29-y²)] / 2

再看第二项。令：For the second term, let:

a = y b = √(29-x²)

则：Then:

y√(29-x²) ≤ [y² + (29-x²)] / 2

两式相加：Add the two inequalities:

x√(29-y²) + y√(29-x²) ≤ [x² + 29-y² + y² + 29-x²] / 2 = 58 / 2 = 29

题目中左边正好等于 29，所以这里必须取等号：The problem says the left side equals 29, so equality must occur:

x√(29-y²) + y√(29-x²) = 29

取等号的条件是两次都满足 a=b。因此：Equality requires a=b in both applications. Therefore:

x = √(29-y²) y = √(29-x²)

由第一个等式平方：Squaring the first equality gives:

x² = 29 - y²

所以：So:

x² + y² = 29
答案：x²+y²=29Answer: x²+y²=29
