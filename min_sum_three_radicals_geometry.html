一、题目1. Problem

题目原图：已知 \(x+y+z=6\)，求根式和的最小值。Original image: Given \(x+y+z=6\), find the minimum of the radical sum.
\[ x+y+z=6 \]
\[ \text{求 }\sqrt{x^2+1}+\sqrt{y^2+9}+\sqrt{z^2+16}\text{ 的最小值。} \]

二、答案2. Answer

\[ \boxed{10} \]
最小值为 \(10\)，当 \(x=\frac34,\ y=\frac94,\ z=3\) 时取得。The minimum is \(10\), achieved at \(x=\frac34,\ y=\frac94,\ z=3\).

三、把每一项看成斜边3. Interpret Each Term as a Hypotenuse

三个根式可以看成三个直角三角形的斜边：Each radical can be interpreted as the hypotenuse of a right triangle:

\[ \sqrt{x^2+1}=\sqrt{x^2+1^2} \]
\[ \sqrt{y^2+9}=\sqrt{y^2+3^2} \]
\[ \sqrt{z^2+16}=\sqrt{z^2+4^2} \]

所以它们分别对应三段折线，水平长度为 \(x,y,z\)，竖直长度为 \(1,3,4\)。They correspond to three path segments with horizontal lengths \(x,y,z\) and vertical lengths \(1,3,4\).

四、图示解法：折线长度 ≥ 直线长度4. Visual Method: Broken Line ≥ Straight Line

三段折线的总水平长度是：The total horizontal length is:

\[ x+y+z=6 \]

总竖直长度是：The total vertical length is:

\[ 1+3+4=8 \]

因此起点到终点的直线距离是：Therefore, the straight distance from the start to the end is:

\[ \sqrt{6^2+8^2}=10 \]
折线长度不小于直线长度，所以原式 \(\ge 10\)。A broken line is at least as long as the straight line, so the expression is at least \(10\).

五、什么时候能取到等号？5. When Does Equality Hold?

等号成立时，三段斜边要排成同一条直线，也就是三个小三角形的斜率相同：Equality holds when the three hypotenuses lie on the same straight line, so the three small triangles have the same slope:

\[ \frac{x}{1}=\frac{y}{3}=\frac{z}{4}=k \]

于是：Thus:

\[ x=k,\qquad y=3k,\qquad z=4k \]

代入 \(x+y+z=6\)：Substitute into \(x+y+z=6\):

\[ k+3k+4k=6 \]
\[ 8k=6 \]
\[ k=\frac34 \]

所以：Therefore:

\[ x=\frac34,\qquad y=\frac94,\qquad z=3 \]
这个取值能让三段折线刚好变成一条直线，所以最小值 \(10\) 可以取到。These values make the broken path become a straight line, so the minimum \(10\) is attainable.

六、代数法：闵可夫斯基不等式6. Algebraic Method: Minkowski's Inequality

由闵可夫斯基不等式：By Minkowski's inequality:

\[ \sqrt{x^2+1^2}+\sqrt{y^2+3^2}+\sqrt{z^2+4^2} \ge \sqrt{(x+y+z)^2+(1+3+4)^2} \]

代入条件：Substitute the given condition:

\[ \sqrt{x^2+1}+\sqrt{y^2+9}+\sqrt{z^2+16} \ge \sqrt{6^2+8^2}=10 \]

当三组向量 \((x,1),(y,3),(z,4)\) 同方向时取等号。Equality holds when the three vectors \((x,1),(y,3),(z,4)\) point in the same direction.

七、最终答案7. Final Answer

\[ \boxed{10} \]
最小值为 \(10\)，当 \(x=\frac34,\ y=\frac94,\ z=3\) 时取得。The minimum is \(10\), achieved at \(x=\frac34,\ y=\frac94,\ z=3\).
