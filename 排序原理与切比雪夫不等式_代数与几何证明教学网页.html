排序原理与切比雪夫不等式

先用代数方法严格证明，再用“长方形面积”直观理解：为什么大数配大数时乘积和更大，以及切比雪夫不等式为何成立。

一、知识总览

排序原理

当两组数都按从大到小排列时：

a1b1 + a2b2 + ··· + anbn

是所有配对方式中最大的乘积和。

切比雪夫不等式

当两组数同序时：

(1/n)Σaibi ≥ [(1/n)Σai][(1/n)Σbi]

即：乘积的平均值不小于两个平均值的乘积。

两个结论的共同核心都是： (ai − aj)(bi − bj) ≥ 0。
代数证明 1

二、排序原理的代数证明

1. 二元情况

设

a1 ≥ a2，　b1 ≥ b2

比较同序配对与交叉配对：

S同 = a1b1 + a2b2
S交 = a1b2 + a2b1

两式相减：

S同 − S交 = (a1−a2)(b1−b2) ≥ 0

因此：

a1b1 + a2b2 ≥ a1b2 + a2b1
大数配大数、小数配小数，乘积和不小于交叉配对。

2. 一般情况：交换法

若某个排列中出现逆序：较大的 a 配了较小的 b，较小的 a 配了较大的 b。交换这两个 b 后，乘积和的增加量为：

(aibp + ajbq) − (aibq + ajbp) = (ai−aj)(bp−bq) ≥ 0

所以每消除一个逆序，乘积和都不会减小。不断交换，最终得到同序排列，因此同序排列最大；反序排列最小。

Σaibn+1−i ≤ Σaibσ(i) ≤ Σaibi
代数证明 2

三、切比雪夫不等式的代数证明

设两组数同序：

a1 ≥ a2 ≥ ··· ≥ an， b1 ≥ b2 ≥ ··· ≥ bn

要证明：

(1/n)Σaibi ≥ [(1/n)Σai][(1/n)Σbi]

方法一：两两作差

因为两组数同序，所以对任意 i、j：

(ai−aj)(bi−bj) ≥ 0

对所有 i、j 求和并展开：

2nΣaibi − 2(Σai)(Σbi) ≥ 0

两边除以 2：

nΣaibi ≥ (Σai)(Σbi)

再除以 n²，即得：

(1/n)Σaibi ≥ [(1/n)Σai][(1/n)Σbi]

方法二：利用排序原理

固定 a 序列，将 b 序列作 n 个循环排列。根据排序原理，每一个循环排列的乘积和都不超过同序排列。

把这 n 个不等式相加。右边中，每个 ai 都恰好与每个 bj 相乘一次，因此：

nΣaibi ≥ (Σai)(Σbi)

这就是切比雪夫不等式。

二元特例： 当 n=2 时，切比雪夫不等式等价于 (a1−a2)(b1−b2)≥0。
几何证明 1

四、排序原理的几何证明

1. 先看二元情况

设

a1 ≥ a2，　b1 ≥ b2

要比较：

同序配对

S同 = a1b1 + a2b2
大配大，小配小

交叉配对

S交 = a1b2 + a2b1
大配小，小配大
1
把每个乘积看成长方形面积。
2
比较同序配对和交叉配对。两者有很多公共面积。
3
去掉公共面积后，同序配对正好多出右上角的橙色长方形。
S同 − S交 = (a1 − a2)(b1 − b2) ≥ 0

因此：

a1b1 + a2b2 ≥ a1b2 + a2b1
几何意义：同序配对比交叉配对多出的量，就是一个长为 a1−a2、宽为 b1−b2 的长方形面积。

2. 推广到一般情况

若某个配对中出现“逆序”：较大的 a 配了较小的 b，而较小的 a 配了较大的 b，那么交换这两个 b 后，乘积和增加：

增加量 = (ai − aj)(bp − bq) ≥ 0

不断消除逆序，最后得到同序排列。因此同序排列最大，反序排列最小。

Σaibn+1−i ≤ Σaibσ(i) ≤ Σaibi

其中 σ 表示任意排列。

几何证明 2

五、切比雪夫不等式的几何证明

设两组数同序：

a1 ≥ a2 ≥ ··· ≥ an， b1 ≥ b2 ≥ ··· ≥ bn

1. 画出总面积

画一个总长为 Σai、总高为 Σbi 的大长方形。它的面积为：

(Σai)(Σbi)

2. 比较对称位置

对任意 i < j，观察四个小长方形：

aibi，aibj， ajbi，ajbj

由于两组数同序，根据排序原理的二元情况：

aibi + ajbj ≥ aibj + ajbi

多出的面积为：

(ai − aj)(bi − bj)

3. 把所有面积差相加

对所有 i < j 的面积差相加：

Σi<j(ai−aj)(bi−bj) ≥ 0

而这个总面积恰好等于：

nΣaibi − (Σai)(Σbi)

所以：

nΣaibi ≥ (Σai)(Σbi)

两边除以 n²，得到切比雪夫不等式：

(1/n)Σaibi ≥ [(1/n)Σai][(1/n)Σbi]

六、n = 3 时的具体展开

当 n = 3 时：

3(a1b1 + a2b2 + a3b3) − (a1+a2+a3) (b1+b2+b3)

可以分解为三个长方形面积：

(a1−a2)(b1−b2) + (a1−a3)(b1−b3) + (a2−a3)(b2−b3)
面积长宽非负原因
S12a1−a2b1−b2同序
S13a1−a3b1−b3同序
S23a2−a3b2−b3同序
因为这三个量都是长方形面积，所以它们都不小于 0，它们的和也不小于 0。

七、排序原理与切比雪夫不等式的关系

排序原理

研究一次交换：

(ai−aj)(bi−bj)

说明消除一个逆序后，乘积和不会减小。

切比雪夫不等式

把所有两两比较的面积加起来：

Σi<j(ai−aj)(bi−bj)

得到总体平均关系。

一句话记忆：
排序原理是“交换一个逆序”；切比雪夫不等式是“把所有逆序比较的面积差加起来”。
