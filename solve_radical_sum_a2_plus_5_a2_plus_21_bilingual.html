一、题目1. Problem

题目原图Original problem image
\[ \sqrt{a^2+5}+\sqrt{a^2+21}=8 \]
\[ \text{求 }a+3\text{ 的值} \]

二、答案2. Answer

\[ \boxed{a+3=1\text{ 或 }5} \]
若 \(a\) 是实数，则 \(a=\pm2\)，所以 \(a+3=1\) 或 \(5\)。If \(a\) is real, then \(a=\pm2\), so \(a+3=1\) or \(5\).
注意：题目只出现 \(a^2\)，所以不能唯一确定 \(a\) 的正负。若额外规定 \(a>0\)，则答案是 \(5\)。 Note: Since only \(a^2\) appears, the sign of \(a\) is not uniquely determined. If an extra condition \(a>0\) is given, the answer is \(5\).

三、换元：令 \(t=a^2\)3. Substitute: Let \(t=a^2\)

因为式子里只出现 \(a^2\)，所以令：Since only \(a^2\) appears, let:

\[ t=a^2,\qquad t\ge0 \]

原方程变为：The equation becomes:

\[ \sqrt{t+5}+\sqrt{t+21}=8 \]

四、第一次平方4. Square the Equation Once

\[ \left(\sqrt{t+5}+\sqrt{t+21}\right)^2=8^2 \]
\[ (t+5)+(t+21)+2\sqrt{(t+5)(t+21)}=64 \]
\[ 2t+26+2\sqrt{(t+5)(t+21)}=64 \]
\[ \sqrt{(t+5)(t+21)}=19-t \]
因为左边是根号，必须有 \(19-t\ge0\)，即 \(t\le19\)。 Since the left side is a square root, we need \(19-t\ge0\), so \(t\le19\).

五、第二次平方5. Square Again

\[ (t+5)(t+21)=(19-t)^2 \]

展开两边：Expand both sides:

\[ t^2+26t+105=t^2-38t+361 \]
\[ 64t=256 \]
\[ t=4 \]

也就是：That is:

\[ a^2=4 \]
\[ a=\pm2 \]

六、求 \(a+3\)6. Find \(a+3\)

当 \(a=2\) 时：When \(a=2\):

\[ a+3=2+3=5 \]

当 \(a=-2\) 时：When \(a=-2\):

\[ a+3=-2+3=1 \]
所以如果 \(a\) 没有限制正负，答案有两个：\(1\) 和 \(5\)。 So if the sign of \(a\) is not restricted, there are two answers: \(1\) and \(5\).

七、验算7. Check

无论 \(a=2\) 还是 \(a=-2\)，都有 \(a^2=4\)，代入原式：Whether \(a=2\) or \(a=-2\), we have \(a^2=4\). Substitute into the original equation:

\[ \sqrt{4+5}+\sqrt{4+21} =\sqrt9+\sqrt{25} =3+5=8 \]

所以两个值都符合原方程。So both values satisfy the original equation.

八、方法二：设差法8. Method 2: Set the Difference

你提供的设差法手写解法 Your handwritten solution using the difference method

这个方法比连续平方更简洁。先设两个根式的和与差。 This method is shorter than squaring repeatedly. First set the sum and the difference of the two radicals.

\[ \sqrt{a^2+5}+\sqrt{a^2+21}=8 \tag{1} \]

设： Let:

\[ \sqrt{a^2+21}-\sqrt{a^2+5}=t \tag{2} \]

把式 (1) 和式 (2) 相乘，左边是平方差： Multiply equations (1) and (2). The left side is a difference of squares:

\[ \left(\sqrt{a^2+21}+\sqrt{a^2+5}\right) \left(\sqrt{a^2+21}-\sqrt{a^2+5}\right) =8t \]
\[ (a^2+21)-(a^2+5)=8t \]
\[ 16=8t \]
\[ t=2 \]

所以： Therefore:

\[ \sqrt{a^2+21}-\sqrt{a^2+5}=2 \tag{3} \]

现在把式 (1) 和式 (3) 相加： Now add equations (1) and (3):

\[ 2\sqrt{a^2+21}=10 \]
\[ \sqrt{a^2+21}=5 \]
\[ a^2+21=25 \]
\[ a^2=4 \]
\[ a=\pm2 \]

因此： Thus:

\[ a+3=5\quad\text{或}\quad a+3=1 \]
所以答案仍然是 \(\boxed{1\text{ 或 }5}\)。如果题目额外规定 \(a>0\)，则答案为 \(\boxed{5}\)。 So the answer is still \(\boxed{1\text{ or }5}\). If the problem additionally states \(a>0\), then the answer is \(\boxed{5}\).

九、最终答案9. Final Answer

\[ \boxed{a+3=1\text{ 或 }5} \]
若题目隐含 \(a>0\)，则 \(\boxed{5}\)。否则答案是 \(\boxed{1\text{ 或 }5}\)。If the problem implicitly assumes \(a>0\), then \(\boxed{5}\). Otherwise, the answer is \(\boxed{1\text{ or }5}\).
