杨辉三角为什么给出 (a+b)n 的系数？

用动画看清楚：每一行的“中间数 = 上一行相邻两数之和”，正好来自多项式乘法中同类项的合并。

1. 先看目标

(a+b)n 展开后，每一项前面的数字就是第 n 行的数字。

例如第 4 行是 1, 4, 6, 4, 1，所以 (a+b)4 的五个系数就是 1, 4, 6, 4, 1。这里从第 0 行开始数：

第 0 行：1；第 1 行：1, 1；第 2 行：1, 2, 1；第 3 行：1, 3, 3, 1。

2. 动画生成杨辉三角

规律：两边永远是 1；中间每个数等于上一行左上、右上两个数的和。

3. 用它写出展开式

4. 数学证明：为什么这个规律正确？

Step 1先看已经知道的一行

第 3 行是 1, 3, 3, 1，所以

(a+b)3 = a3 + 3a2b + 3ab2 + b3

每一项前面的数字，就是这一行里的数字。现在我们要看：乘上一个新的 (a+b) 后，下一行为什么会变成 1, 4, 6, 4, 1。

Step 2每一项都会分成两项

因为 (a+b)4 = (a+b)3(a+b)，所以第 3 行里的每一项都要分别乘 a 和 b。

3a2b · a = 3a3b
3a2b · b = 3a2b2

也就是说，一个旧项会给下一行送出两份贡献：一份乘 a，一份乘 b。

Step 3中间的数来自左右两边相加

展开时，同样的字母项要合并。比如 a2b2 可以从两条路来：

  • 上一行的 3a2b 乘以 b，得到 3a2b2。
  • 上一行的 3ab2 乘以 a，也得到 3a2b2。
3a2b2 + 3a2b2 = 6a2b2

所以第 4 行中间的 6，就是上一行相邻两个 3 相加得到的。

Step 4两边为什么还是 1？

最左边的 a4 只能由 a3 乘 a 得到，没有第二条路，所以前面的数字是 1。

最右边的 b4 只能由 b3 乘 b 得到，也没有第二条路，所以前面的数字也是 1。

因此：下一行的两边是 1；中间每个数都是上一行相邻两个数相加。这个规则每次从一行走到下一行都成立，所以杨辉三角正好给出 (a+b)n 的系数。

5. 用“每个括号选 a 或 b”解释每一项怎样形成

下面把 (a+b)4 看成四个括号相乘。每次从每个括号里选出一个字母：选了几个 a、几个 b，就会形成对应的项；同类选择的总个数，就是这一项前面的系数。

所以：a4 只有 1 种选法，a3b 有 4 种，a2b2 有 6 种，ab3 有 4 种，b4 有 1 种。于是系数正好是 1, 4, 6, 4, 1。

6. 题目中的结果

(a+b)4 = a4 + 4a3b + 6a2b2 + 4ab3 + b4
(a+b)5 = a5 + 5a4b + 10a3b2 + 10a2b3 + 5ab4 + b5

用多项式乘法验证 (a+b)4

已知 (a+b)3 = a3 + 3a2b + 3ab2 + b3，两边乘 (a+b)：

(a3 + 3a2b + 3ab2 + b3)(a+b)
= a4 + a3b + 3a3b + 3a2b2 + 3a2b2 + 3ab3 + ab3 + b4
= a4 + 4a3b + 6a2b2 + 4ab3 + b4

中间的 4, 6, 4 正好来自上一行 1, 3, 3, 1 的相邻相加：1+3=4，3+3=6，3+1=4。
