一、题目1. Problem

已知 x⁵+x+1=0，求 x³−x² 的值。Given x⁵+x+1=0, find x³−x².
已知：x⁵ + x + 1 = 0 求：x³ - x²

二、关键：因式分解2. Key: Factorization

不要直接“看出”因式分解，可以先添项、减项。也就是把 0 拆成几组：Instead of just guessing the factorization, add and subtract terms. We split 0 into several pairs:

x⁵ + x + 1 = x⁵ - x⁴ + x⁴ - x³ + x³ + x² - x² + x + 1

然后按三项一组重新分组：Then group the terms in groups of three:

x⁵ - x⁴ + x² + x⁴ - x³ + x + x³ - x² + 1

每一组都能提出同一个因式：Each group has the same common factor:

x⁵ - x⁴ + x² = x²(x³ - x² + 1) x⁴ - x³ + x = x(x³ - x² + 1) x³ - x² + 1 = 1·(x³ - x² + 1)

所以：Therefore:

x⁵ + x + 1 = x²(x³ - x² + 1) + x(x³ - x² + 1) + (x³ - x² + 1) = (x² + x + 1)(x³ - x² + 1)

三、实数条件下求值3. Evaluate Under the Real-Number Assumption

通常初中题默认 x 为实数。对实数 x：In middle-school problems, x is usually assumed to be real. For real x:

x² + x + 1 = (x + 1/2)² + 3/4 > 0

所以只能：Therefore:

x³ - x² + 1 = 0

移项得到：Rearrange:

x³ - x² = -1
答案：-1Answer: -1

四、补充说明4. Note

如果允许复数解，x²+x+1=0 也有解，此时 x³−x² 不唯一。所以本题要得到唯一答案，需要默认 x 是实数。If complex roots are allowed, x²+x+1=0 also has roots, and x³−x² is not unique. A unique answer requires the usual assumption that x is real.

五、最终答案5. Final Answer

x³ - x² = -1
