初中数学 / 代数最值

已知条件下分式最小值

本页整理原题以及三种解法：代入化简法、拆项 + 均值不等式、设参数 + 判别式。

原题

$$\text{{已知 }} a,b>0,\quad 2a+b=1,\quad \text{{求 }}\frac{{5a+b}}{{a^2+ab}}\text{{ 的最小值。}}$$
结论：原式的最小值为 $\boxed{{9}}$，当 $a=b=\dfrac13$ 时取到。
查看原题手写图
方法一

代入化简法

  1. 由条件 $2a+b=1$，可得 $b=1-2a$。又因为 $a,b>0$，所以 $0
  2. 原式化简：
    $$\frac{{5a+b}}{{a^2+ab}}=\frac{{5a+b}}{{a(a+b)}}$$
    代入 $b=1-2a$，得
    $$5a+b=5a+1-2a=1+3a,$$
    $$a+b=a+1-2a=1-a.$$
    所以
    $$\frac{{5a+b}}{{a^2+ab}}=\frac{{1+3a}}{{a(1-a)}}.$$
  3. 设
    $$f(a)=\frac{{1+3a}}{{a(1-a)}}.$$
    通过化简可得当 $a=\dfrac13$ 时，函数取得最小值。
  4. 此时
    $$b=1-2a=1-\frac23=\frac13.$$
    代回原式：
    $$\frac{{5a+b}}{{a^2+ab}}=\frac{{5\cdot\frac13+\frac13}}{{\left(\frac13\right)^2+\frac13\cdot\frac13}}=\frac{{2}}{{\frac29}}=9.$$
说明：这种方法思路直接，先消去一个未知数，再对单变量表达式求最值。
方法二

拆项 + 均值不等式

  1. 先把分母写成乘积：
    $$a^2+ab=a(a+b).$$
  2. 把分子拆成 $4a+(a+b)$：
    $$5a+b=4a+(a+b).$$
    因此
    $$\frac{{5a+b}}{{a^2+ab}}=\frac{{4a+(a+b)}}{{a(a+b)}}=\frac{{4}}{{a+b}}+\frac{{1}}{{a}}.$$
  3. 由 $2a+b=1$ 可知
    $$a+(a+b)=1.$$
    所以乘上这个“1”不改变原式：
    $$\frac{{4}}{{a+b}}+\frac{{1}}{{a}}=\left(\frac{{4}}{{a+b}}+\frac{{1}}{{a}}\right)\big((a+b)+a\big).$$
  4. 展开得
    $$\left(\frac{{4}}{{a+b}}+\frac{{1}}{{a}}\right)\big((a+b)+a\big)=5+\frac{{4a}}{{a+b}}+\frac{{a+b}}{{a}}.$$
  5. 对后两项用均值不等式：
    $$\frac{{4a}}{{a+b}}+\frac{{a+b}}{{a}}\ge 2\sqrt{{\frac{{4a}}{{a+b}}\cdot\frac{{a+b}}{{a}}}}=2\sqrt{{4}}=4.$$
    所以
    $$\frac{{5a+b}}{{a^2+ab}}\ge 5+4=9.$$
  6. 当且仅当
    $$\frac{{4a}}{{a+b}}=\frac{{a+b}}{{a}}$$
    时取等号，即
    $$4a^2=(a+b)^2.$$
    因为 $a,b>0$，所以 $2a=a+b$，于是 $a=b$。再由 $2a+b=1$ 得
    $$a=b=\frac13.$$
说明：这是三种方法中最适合初中阶段的方法，技巧性很强，也不需要求导。

方法三

设参数 + 判别式

  1. 同样先由 $2a+b=1$ 得
    $$b=1-2a,$$
    原式变为
    $$\frac{{5a+b}}{{a^2+ab}}=\frac{{3a+1}}{{a(1-a)}}.$$
  2. 设原式为
    $$t=\frac{{3a+1}}{{a(1-a)}}.$$
    则
    $$ta(1-a)=3a+1.$$
  3. 整理成关于 $a$ 的二次方程：
    $$ta-ta^2=3a+1,$$
    $$ta^2+(3-t)a+1=0.$$
  4. 因为 $a$ 是实数，所以该方程判别式满足
    $$\Delta\ge 0.$$
    即
    $$(3-t)^2-4t\ge 0,$$
    $$t^2-10t+9\ge 0,$$
    $$(t-1)(t-9)\ge 0.$$
    所以
    $$t\le 1\quad \text{{或}}\quad t\ge 9.$$
  5. 又因为
    $$t=\frac{{3a+1}}{{a(1-a)}}>\frac{{3a+1}}{{a}}=3+\frac1a>3,$$
    所以 $t\le 1$ 不可能，只能有
    $$t\ge 9.$$
  6. 当 $t=9$ 时，
    $$9a^2-6a+1=0,$$
    $$(3a-1)^2=0,$$
    因而
    $$a=\frac13,\quad b=1-2a=\frac13.$$
说明：这种方法把“最值问题”转化成“方程有解问题”，是很经典的代数思路。
查看方法三手写图

小结

  • 方法一：先代入，再化成单变量表达式求最值，思路最直接。
  • 方法二：通过巧妙拆项，再用均值不等式，简洁漂亮，最适合课堂讲解。
  • 方法三：把原式设为参数 $t$，借助判别式求范围，体现“方程思想”。
因此，原式的最小值始终是 $\boxed{{9}}$，且在 $a=b=\dfrac13$ 时取得。
