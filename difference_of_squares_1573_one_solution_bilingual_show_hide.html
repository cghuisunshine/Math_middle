一、题目1. Problem

题目原图：a、b 都是正整数，a²-b²=1573，求 a+b 的一组值。 Original image: a and b are positive integers, a²-b²=1573. Find one possible value of a+b.
a² - b² = 1573 a, b 都是正整数 求 a+b = ?（求一组解）

二、解法：平方差公式2. Solution: Difference of Squares

利用平方差公式：Use the difference of squares identity:

a² - b² = (a-b)(a+b)

所以：So:

(a-b)(a+b)=1573

分解 1573：Factor 1573:

1573 = 11 × 143

因此可以取：So we can choose:

a-b = 11 a+b = 143

解出 a、b：Solve for a and b:

2a = 154 → a = 77 2b = 132 → b = 66

检验：Check:

77² - 66² = (77-66)(77+66) = 11 × 143 = 1573
一组解：a=77，b=66，所以 a+b=143One solution: a=77, b=66, so a+b=143

三、补充：答案不唯一3. Note: The Answer Is Not Unique

因为题目说“求一组解”，所以只要给出一组满足条件的正整数即可。实际上还有其他因数分组：Because the problem asks for one solution, it is enough to give any valid positive-integer pair. There are other factor pairings:

1573 = 1 × 1573 1573 = 11 × 143 1573 = 13 × 121

对应的 a+b 可以是：Corresponding possible values of a+b include:

1573, 143, 121
如果题目要求“最小的 a+b”，则应选 121；如果只要求“一组解”，143 就可以。 If the problem asks for the smallest a+b, choose 121. If it asks for just one solution, 143 is valid.

四、最终答案4. Final Answer

a=77, b=66 a+b=143
最终答案：143（一组解）Final answer: 143 (one possible solution)
